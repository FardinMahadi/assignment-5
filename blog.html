What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web pages. It represents the structure of an HTML or XML document as a tree of objects (nodes), where each element or content is a node.

How do you select an element from the DOM?

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the DOM where a single event listener is attached to a parent element to manage events triggered by its child elements. Instead of adding individual event listeners to multiple child elements, the parent listens for events that "bubble up" from the children. This works because most events in the DOM bubble up from the target element to its parents.

How do you manipulate an element's attributes and styles using the DOM?

To manipulate an element's attributes and styles in the DOM, you can use JavaScript. For attributes, you can use setAttribute() to add or modify an attribute, getAttribute() to retrieve its value, and removeAttribute() to remove it. For styles, you can directly modify the element's style property by setting specific CSS properties, such as backgroundColor or fontSize. This allows you to dynamically adjust the appearance of elements or change attributes like href or src, providing flexibility in how elements behave or look on the webpage.
